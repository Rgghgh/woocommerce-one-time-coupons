Adds Single use coupons to woocommerce.
Allows you to generate multiple single-use codes for a single woocommerce coupon.

== Description ==

This plugin will allow you to generate multiple, one-time use, codes for a woocommerce coupon.
Example usage: You want to sell coupons through a third-party, and want each coupon to work only once.

== Screenshots ==

1. Adds options to each woocommerce coupon
2. Set a prefix, and the amount of codes to generate. Keep track of used and free coupons.
